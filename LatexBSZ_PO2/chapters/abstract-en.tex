\chapter*{Abstract}
\label{cha:abstract-en}

The documentation presents the ``Rock Gym'' project, designed for the management of a climbing wall by employees and managers. The main goal of the project was to create a tool that streamlines the work of the staff and increases customer convenience by introducing a QR code and fingerprint scanner. The solution consists of an application written in C\# using the .NET framework, which communicates with a relational database.

As part of the work, modules responsible for managing the customer and staff database, handling passes, events, notifications, and a biometric identity verification system for users entering the facility were implemented. The biometric part of the project involves retrieving data from image files, extracting features, and comparing fingerprint patterns while maintaining appropriate security standards.
